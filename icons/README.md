# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels (toolbar)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Option: Use Emoji-based Icons

For development, you can create simple icons using any image editor or online tool:

1. Go to https://favicon.io/emoji-favicons/
2. Search for "magnifying glass" or "shield" emoji
3. Download the favicon pack
4. Rename the files to icon16.png, icon48.png, icon128.png
